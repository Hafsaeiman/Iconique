# ICONIQUE – Luxury Fashion E-Commerce Website

## Overview

Iconique is a modern fashion e-commerce platform designed to deliver a seamless and elegant online shopping experience. The website showcases premium fashion collections for Women, Men, and Kids while providing an intuitive user interface built around Human-Computer Interaction (HCI) principles and usability standards.

## Features

### Customer Features

* User Registration & Login
* Browse Women, Men, and Kids Collections
* Product Search Functionality
* Advanced Product Filtering
* Product Detail Pages
* Wishlist Management
* Shopping Cart
* Secure Multi-Step Checkout
* Order Summary & Shipping Details
* Customer Reviews & Ratings
* Contact Form

### Design & UX Features

* Responsive Layout
* Clean Visual Hierarchy
* Consistent Navigation
* Accessibility-Oriented Design
* HCI Laws and Usability Principles Applied
* Modern Fashion-Focused Interface

## HCI Principles Implemented

* Jakob’s Law
* Hick’s Law
* Fitts’s Law
* Miller’s Law
* Law of Proximity
* Law of Similarity
* Common Region Principle
* Aesthetic-Usability Effect
* Visibility of System Status
* Consistency & Standards
* User Control & Freedom
* Error Prevention

## Technologies Used

* Figma (UI/UX Design)
* HTML5
* CSS3
* JavaScript
* Responsive Design Principles

## Project Goals

The primary goal of Iconique is to create a visually appealing, user-friendly, and conversion-focused fashion shopping platform that enhances customer satisfaction while maintaining a premium brand identity.
